WINDOWS OS

WINDOWS 95:WINDOWS 95 was the first windows OS.It was released in 1995.It was compatible with MSDOS hardware.It started to be manufactured on July 14, 1995.

WINDOWS 98:it was the successor to Windows 95, and was released to manufacturing on May 15, 1998.

WINDOWS 2000:It was the successor to Windows 98.It was released by MICROSOFT in early February 2000.Support for this OS was discontinued in 2010.

WINDOWS XP:It was a popular OS released by MICROSOFT. It started to be manufactured on August 24, 2001.

WINDOWS Vista:It was an OS that wasn't popular among users because of compatibility issues.It started to be manufactured on 30 JANUARY 2007.

WINDOWS 7:It was a WINDOWS OS that was a huge hit.It had almost the same UI as VISTA.It started to be manufactured on July 22, 2009.

WINDOWS 8:It was a WINDOWS OS that was great to run on a tablet or a smartphone than on a desktop or laptop.It started to be manufactured on August 1, 2012.

WINDOWS 10: This is an OS that I liked to use.It started to be manufactured on 10 in July 2015 .Its official end of support date is October 2025.

WINDOWS 11:This OS is the successor to WINDOWS 10.Its manufacture date is October 5, 2021.
